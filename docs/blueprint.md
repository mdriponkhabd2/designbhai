# **App Name**: DesignBhai Admin Panel

## Core Features:

- Single Admin Login: A secure, password-only login system allowing the website owner to access the admin dashboard. No signup functionality is provided; access is granted upon correct password entry and denied otherwise.
- Portfolio Content Management: Admins can upload new portfolio images, add titles and categories to them, edit existing image details, and delete portfolio items. Changes made here automatically reflect on the main 'DesignBhai' website's portfolio section.
- Services Content Management: Provides functionality to add new service offerings, edit descriptions and details of existing services, and remove services from the website.
- About Section Content Management: Allows the admin to update the 'About Us' image and edit the descriptive text displayed in the 'About Us' section of the main website.
- Contact Information Management: Enables the admin to easily edit and update the primary phone numbers and physical address displayed in the 'Contact Us' section of the main website.
- Website Global Settings: Admins can update fundamental website settings such as changing the favicon and modifying the site's main title.
- Dashboard Navigation: A sidebar navigation menu providing quick access to different administrative sections: Dashboard, Portfolio, Services, About, Contact, and Settings.

## Style Guidelines:

- Clean, modern dashboard layout with a clear sidebar navigation menu for easy access to different management sections.
- Fully mobile-responsive admin interface ensuring usability across various screen sizes (desktop, tablet, mobile).
- A professional and subdued color palette, possibly using muted greens, grays, and white to maintain a clean aesthetic, while subtly referencing the main website's branding.
- Modern, sans-serif typography for readability and a contemporary feel, ensuring clarity for administrative tasks.
- Simple and intuitive icons to represent various management actions (e.g., upload, edit, delete) and navigation items, enhancing the user experience.
